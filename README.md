# Skills Collection

个人插件市场，收录和总结好用的 Claude Code 技能插件集合。

## 插件列表

| 插件名称 | 描述 | 技能数量 |
|---------|------|---------|
| [leetcode](./leetcode/) | LeetCode 算法题解助手 | 1 |

## 使用方式

将所需插件的技能文件（`.md`）添加到你的 Claude Code 配置中即可使用。

## 目录结构

```
skills-collection/
├── README.md
└── leetcode/
    ├── README.md
    └── leetcode-solver.md
```
